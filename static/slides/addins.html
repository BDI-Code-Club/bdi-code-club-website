WHAT ARE RSTUDIO ADD-INS?

What are RStudio add-ins?

  • A mechanism for executing R functions interactively from within the RStudio IDE

  • Addins can facilitate many tasks including:
    • writing r markdown documents
    • styling your R code
    • copying tables to and from Rstudio
    • interactively choosing colours for your graphs

Installed as R packages

# Install add-in "remedy" from CRAN:
install.packages("remedy")

# Install add-in "WrapRmd" from GitHub:
devtools::install_github("tjmahr/WrapRmd")

Appear in the Addins drop down menu
